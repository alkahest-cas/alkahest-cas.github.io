# alkahest-cas.github.io

Organization landing page for [Alkahest](https://github.com/alkahest-cas/alkahest).

- **Edit:** `src/index.html`, `src/styles.css`
- **Publish:** run `bun run build.ts` to copy artifacts into the repo root (`index.html`, `styles.css`), then commit and push `main` (GitHub Pages serves from this repo).

Full documentation and install guide (PyPI default vs optional `+jit` / `+full` wheels, from source): **[alkahest-cas.github.io/alkahest/](https://alkahest-cas.github.io/alkahest/)**
